❓ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

"ANTHROPIC_API_KEY not found"
  → Check .env.local exists
  → Check it has your key: grep ANTHROPIC ../.env.local
  → Copy template if missing: cp ../.env.template ../.env.local

"ModuleNotFoundError: No module named 'anthropic_usage'"
  → Install deps: pip install -r ../requirements.txt
  → Or run from project root: python -m tools.anthropic_usage

"API Error: 401 Unauthorized"
  → Your API key is invalid or expired
  → Generate new key: https://console.anthropic.com/account/keys
  → Update .env.local
